Q 1. What are Pull Requests used for?
Pull request are used to submit our changes for the someone to check and accept it or not.

Q 2. Name 2 other examples of Version Control Systems apart from Git.
Git Hub and Subversion


Q 3. What is a repo and explain the difference between a local and remote one? 
Repo is a repository of files that are part of a project. Local ones are in our devices and remote ones are in version control systems like Git Hub.

Q 4. Where would you find a local repo?
Local repo is in our computer (where we cloned the master files).
